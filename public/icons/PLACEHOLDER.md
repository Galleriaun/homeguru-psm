# PWA icons go here

Generate and drop in this folder before going to production:

| File | Size | Purpose |
|---|---|---|
| `192.png` | 192×192 | Android home screen |
| `512.png` | 512×512 | Android splash + larger displays |
| `maskable-512.png` | 512×512 | Maskable variant for Android adaptive icons |
| `apple-touch-180.png` | 180×180 | iOS home screen |

**Tools:**
- [maskable.app/editor](https://maskable.app/editor)
- [realfavicongenerator.net](https://realfavicongenerator.net/)

Delete this file once icons are in place.
